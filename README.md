## 📌 Overview

This project demonstrates an AI-powered email automation workflow built using **n8n** and **OpenAI**. The workflow generates professional email drafts from natural language prompts, maintains conversational context using memory, and sends emails through Gmail after user approval.

---

## 🚀 Features

- AI-powered email generation
- Chat-based interaction
- Context-aware conversations using Simple Memory
- User approval before sending emails
- Gmail integration
- Event-driven workflow automation

## 🏗️ Workflow Architecture

```text

   │
   ▼

   │
   ▼

   │
   ▼
OpenAI Chat Model
   │
   ▼
Simple Memory
   │
   ▼
Generate Email Draft
   │
   ▼
User Approval
   │
   ▼
Gmail
```
